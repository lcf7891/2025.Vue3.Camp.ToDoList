## 資料夾結構說明

```
src/
├── assets/       # 資源
│ └── images/     # 圖片
│ └── style/      # 樣式
├── components/   # 元件
├── composables/  # 邏輯封裝
├── router/       # Vue Router 設定
├── stores/       # Pinia 狀態管理
├── views/        # 頁面
├── App.vue       # 根元件
└── main.js       # 進入點
```
